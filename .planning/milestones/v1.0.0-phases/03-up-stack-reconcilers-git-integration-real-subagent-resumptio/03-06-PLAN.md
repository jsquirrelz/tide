cmd/tide-push CLI flags (planner sets exact names; convention: --kebab-case):
- `--mode={clone|push}` — clone-mode = bare clone repo to /workspace/repo.git; push-mode = compute diff + scan + push.
- `--commit-message` — required in push-mode (D-B2 / W11): the controller-computed boundary message (one of 4 fixed shapes). Push Job uses this verbatim as the commit message.
- `--artifact-paths` — required in push-mode: CSV list of paths (relative to /workspace) the push Job stages + commits before push. Empty list = no artifacts to stage (defensive — D-B2 commit always happens, even if it's just metadata).
- `--repo-url` — for clone-mode (mirrors pkg/git.Clone repoURL arg).
- `--branch` — per-run branch (D-B6 format `tide/run-<project-name>-<unix>`).
- `--last-pushed-sha` — for push-mode; empty string on first push.
- `--leaks-config` — optional path to per-Project ConfigMap-mounted gitleaks override TOML.
- `--workspace` — root, default `/workspace`.
- `--project-uid` — for push-result envelope path `/workspace/envelopes/push/{project-uid}.json`.

Required env vars (envFrom git creds Secret):
- `GIT_PAT` — the personal access token. Push Job reads this from env; pkg/git.Push receives it as a string arg.

Exit codes (per 03-RESEARCH Open Questions Q5 RESOLVED; ProjectReconciler in plan 03-08 maps each `reason` to a distinct Status.phase):
- `0` — success (push or clone succeeded). Push-result envelope written with `reason=""`.
- `1` — generic failure (catch-all; reason field carries the specific cause).
- `2` — invariant violation (push-result envelope `reason=invalid-branch` for D-B6 main-guard, or `reason=missing-creds` for absent GIT_PAT env).
- `10` — gitleaks block (`reason=leak-detected`). Reconciler patches `Status.phase=Failed` + `Condition=SecretLeakBlocked`.
- `11` — lease rejection (`reason=lease-rejected`). Reconciler patches `Status.phase=PushLeaseFailed` + `LeaseFailureCount++` (D-B6 recovery via `tideproject.k8s/bypass-push-lease=true` annotation).
- `12` — auth failure (`reason=auth-failed`, e.g., bad/expired PAT). Reconciler patches `Status.phase=Failed` with structured reason.
- `13` — network/timeout (`reason=network-timeout`, e.g., remote unreachable). Reconciler patches `Status.phase=Failed` with structured reason.

Push-result envelope shape (small JSON for ProjectReconciler to read):
```
{
  "apiVersion": "tideproject.k8s/v1alpha1",
  "kind": "PushResult",
  "projectUID": "...",
  "branch": "tide/run-foo-1747200000",
  "headSHA": "abc123...",
  "exitCode": 0,
  "reason": "" | "leak-detected" | "lease-rejected"
}
```
(Mirrors Phase 2 D-A2 envelope-on-PVC contract; the ProjectReconciler reads this in plan 03-08.)
