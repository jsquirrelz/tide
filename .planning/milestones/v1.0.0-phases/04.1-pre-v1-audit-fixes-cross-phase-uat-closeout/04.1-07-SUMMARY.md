### reconcileDispatch (post-extraction, 51 lines)

```go
func (r *TaskReconciler) reconcileDispatch(ctx, task) (ctrl.Result, error) {
    gate, err := r.gateChecks(ctx, task)
    if err != nil { return ctrl.Result{}, err }
    if gate.shouldHalt { return gate.result, nil }

    release, err := r.acquireDispatchSlots(ctx, task, gate.project)
    if err != nil {
        if errors.As(err, &rlErr) { return ctrl.Result{RequeueAfter: rlErr.delay}, nil }
        return ctrl.Result{}, err
    }
    committed := false
    defer func() { if !committed { release() } }()

    spec, err := r.prepareDispatch(ctx, task, gate.project)
    if err != nil {
        if errors.As(err, &maErr) { return ctrl.Result{}, nil }
        return ctrl.Result{}, err
    }

    result, err := r.createDispatchJob(ctx, task, spec)
    if err == nil { committed = true }
    return result, err
}
```

## CR-03 Deferred Reservation Cancel (Preserved)

The pre-extraction code had `committed bool` + `defer` spanning the rate-limit gate and Job create. The extraction preserves this via:

1. `acquireDispatchSlots` captures `heldReservation` in a closure and returns `release func()` that cancels it.
2. `reconcileDispatch` defers `release()` and sets `committed = true` only when `createDispatchJob` succeeds.
3. On any error between `acquireDispatchSlots` and successful `createDispatchJob`, the deferred `release()` returns the token.

Regression test: `TestReconcileDispatch_CommittedReleaseSuppression` â€” verifies the rate-limited sentinel error is translated to `RequeueAfter > 0` with `nil` error (not leaked as an error to callers).

## Sentinel Error Types

Two sentinel types carry clean-halt state across method boundaries without requiring multiple return values:

- `rateLimitedError{delay time.Duration}` â€” returned by `acquireDispatchSlots` when bucket is exhausted; `reconcileDispatch` unwraps to `ctrl.Result{RequeueAfter: d}`.
- `maxAttemptsError{}` â€” returned by `prepareDispatch` after patching Task.Status.Phase=Failed; `reconcileDispatch` returns `ctrl.Result{}, nil` (clean halt, no requeue).
