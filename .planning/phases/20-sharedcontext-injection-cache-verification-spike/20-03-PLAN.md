|----------|-------------|
| LLM (parent planner EnvelopeOut) → controller/materializer | The parent subagent authors SharedContext; the controller treats it as untrusted input before writing to child specs. |
| materializer → child CRD spec (etcd) | SharedContext is persisted to N child object specs; per-object size bounded by etcd (~1.5 MiB). |
| planner path vs executor path | SharedContext must populate planner EnvelopeIn but NEVER the executor EnvelopeIn (CACHE-02). |
