Purpose: This is where the K8s-side dispatch contract becomes real. Per D-B1, TaskReconciler is the SOLE Job creator; per D-B5, the Job name `tide-task-{task-uid}-{attempt-n}` IS the SUB-03 idempotency dedup key (AlreadyExists on duplicate create = success). NOTE ON PVC ACCESS (RESEARCH.md Open Question #2 ‚Äî RESOLVED): Both Manager and Task pods share ONE chart-provisioned `tide-projects` RWX PVC (Blocker #2/#3 resolution). Per-Project isolation is via `subPath: {project-uid}/workspace` (kubelet-enforced). Manager mounts the PVC at `/workspaces` (root, no subPath), so `FilesystemEnvelopeReader{WorkspaceRoot: '/workspaces'}` reads `/workspaces/{project-uid}/workspace/envelopes/{task-uid}/out.json` directly. Task pods mount the same PVC with `subPath: {project-uid}/workspace` so the container sees `/workspace` as its per-Project root. The envelope-writer init container STILL exists (writes EnvelopeIn JSON to PVC before subagent + credproxy start) but it writes via the subPath-mounted `/workspace/envelopes/{task-uid}/in.json` path ‚Äî same PVC as the Manager, just a different mount point. Document this choice in `internal/dispatch/podjob/doc.go`.

```
// BuildJobSpec returns a complete *batchv1.Job for the Task at the given
// attempt number. Caller (TaskReconciler / PodJobBackend) supplies:
//   - task: the Task CRD
//   - project: the owning Project (for ProviderSecretRef + namespace)
//   - attempt: the next attempt number
//   - signedToken: the HMAC-signed token for ANTHROPIC_API_KEY substitution
//   - envelopeInJSON: the marshalled EnvelopeIn that the envelope-writer init container will write to /workspace/envelopes/{taskUID}/in.json
//   - subagentImage, credproxyImage: image refs from Helm values
//   - secretUID: the K8s UID of the Project's providerSecretRef Secret (for tideproject.k8s/provider-secret-uid label)
//
// Returns a Job ready for client.Create.
type BuildOptions struct {
    Task            *tidev1alpha1.Task
    Project         *tidev1alpha1.Project
    Attempt         int
    SignedToken     string
    EnvelopeInJSON  []byte
    SubagentImage   string
    CredproxyImage  string
    SecretUID       string
    PVCName         string  // "tide-projects" ‚Äî single shared PVC for all Projects (Blocker #2/#3); Plan 12 chart provisions it.
    ProjectUID      string  // used as subPath: {project.UID}/workspace under the shared PVC
}
func BuildJobSpec(opts BuildOptions) *batchv1.Job
```
