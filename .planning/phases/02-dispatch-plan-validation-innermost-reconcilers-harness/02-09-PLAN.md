TaskReconciler.Reconcile BODY (inside the `if r.Dispatcher != nil` seam — 12 steps):
1. **Terminal short-circuit:** if `task.Status.Phase == "Succeeded" || "Failed"` → return ctrl.Result{}, nil.
2. **On Running:** if a Job for this Task is in terminal state, run `handleJobCompletion` (reads EnvelopeOut, patches Status, calls RollUpUsage). Return.
3. **Resolve Project:** via `r.resolveProject(ctx, &task)` (uses `tideproject.k8s/project=<name>` label PlanReconciler stamps; falls back to owner-walk on label miss).
4. **Budget gate:** if `project.Status.Phase=BudgetExceeded` AND NOT `budget.IsBypassed(project, time.Now())` → patch Task condition `BudgetBlocked=True`, return.
5. **Indegree compute (D-B3):** `siblings := r.listSiblingTasks(ctx, &task)` (via `.spec.planRef` field-indexer); `indegree := r.computeIndegree(&task, siblings)`. If `indegree > 0` → patch Phase=Pending, return.
6. **Rate-limit gate (Pattern 1):** resolve Secret from `project.Spec.ProviderSecretRef`; `lim := r.Budget.ForSecret(string(secret.UID), r.defaultsForSecret(&secret))`; if `lim != nil`: `rsv := lim.Reserve(); d := rsv.Delay()`. If `d > 0`: `rsv.Cancel()`; `budget.ProviderRateLimitHitsTotal.WithLabelValues(project.Name).Inc()`; return `ctrl.Result{RequeueAfter: d}`.
7. **Attempt counter:** `attempt := r.nextAttempt(ctx, &task)` (list Jobs labeled with this Task UID, max+1). If `attempt > project.Spec.MaxAttemptsPerTask` → Phase=Failed, Reason=ExceededAttempts.
8. **Mint signed token:** `token, _ := credproxy.Sign(r.SigningKey, string(task.UID), time.Duration(task.Spec.Caps.WallClockSeconds + 60)*time.Second)`.
9. **Build EnvelopeIn:** `envIn, _ := r.buildEnvelopeIn(&task, &project, attempt, token)` → marshals JSON; translates api/v1alpha1.Caps to pkgdispatch.Caps per Plan 03's two-type design.
10. **Patch Status.Attempt BEFORE Create (Pitfall 2 mitigation):** `patch := client.MergeFrom(task.DeepCopy()); task.Status.Attempt = int32(attempt); r.Status().Patch(ctx, &task, patch)`.
11. **Build + Create Job:** `opts := podjob.BuildOptions{Task,Project,Attempt,SignedToken,EnvelopeInJSON,SubagentImage,CredproxyImage,SecretUID,PVCName}`; `job := podjob.BuildJobSpec(opts)`; `owner.EnsureOwnerRef(job, &task, r.Scheme)`; `err := r.Create(ctx, job); if err != nil && !apierrors.IsAlreadyExists(err) → return err`.
12. **Patch Status.Phase=Running** + Condition `Running=True, Reason=Dispatched`. Return.

`handleJobCompletion(ctx, task, project)`:
- `out, err := r.EnvReader.ReadOut(ctx, string(project.UID), string(task.UID))` → reads `/workspaces/{project.UID}/workspace/envelopes/{task.UID}/out.json` from the shared `tide-projects` PVC mounted on the Manager pod (Blocker #2/#3 single-PVC architecture; `EnvReader.ReadOut` signature now takes both UIDs). On err → Phase=Failed, Reason=EnvelopeReadFailed.
- **Output-path validation (Warning #5 — wires HARN-05 into dispatch chain):** if `out.Result != "output-paths-violation"` and `task.Spec.DeclaredOutputPaths` is non-empty:
  - Compute `taskWorkspaceRoot := "/workspaces/" + string(project.UID) + "/workspace"` (Manager-side view of the per-Task slice).
  - Call `violations, vErr := outputs.Validate(taskWorkspaceRoot, task.Status.StartedAt.Time, task.Spec.DeclaredOutputPaths)` (note: `internal/harness/outputs.Validate` from Plan 06; works because Manager can see the same files via `/workspaces` mount).
  - If `vErr != nil` → Phase=Failed, Reason=OutputValidationError, log vErr.
  - If `len(violations) > 0` → Phase=Failed, Reason=OutputPathsViolation; record K8s Event listing the first few violations (truncate to keep Event message size manageable).
- Standard result interpretation: if `out.ExitCode != 0 || out.Result == "cap-hit" || out.Result == "output-paths-violation"` → Phase=Failed, Reason=out.Result.
- Else → Phase=Succeeded.
- Patch `Task.Status.CompletedAt = out.CompletedAt`.
- `budget.RollUpUsage(ctx, r.Client, &project, out.Usage)`.

Per Warning #5 + #6: the controller hosts the output-path validator in Phase 2 (deviation from RESEARCH.md Responsibility Map's "Per-Pod subagent" assignment); the harness library at `internal/harness/outputs.go` is imported but invoked controller-side because the Plan 04 stub-subagent runs directly (not wrapped by the harness).
