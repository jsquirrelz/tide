   - Else if `project.Status.Phase != "BudgetExceeded"` AND capExceeded AND NOT bypassed → patch `Project.Status.Phase=BudgetExceeded` + Condition `BudgetExceeded=True, Reason=AbsoluteCapReached` + record K8s Event `Reason=AbsoluteCapReached`.

   - Init Job name: `tide-init-{project.UID}`.
   - Init Job body: single container `busybox` image (Plan 12 Helm value `images.busybox.repository:tag`); command `["sh","-c","mkdir -p /workspace/repo /workspace/artifacts /workspace/envelopes && chmod 0775 /workspace/repo /workspace/artifacts /workspace/envelopes"]`.

   - `ttlSecondsAfterFinished: 300`.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator → bypass annotation | An operator with `kubectl annotate projects` permission can clear BudgetExceeded. RBAC controls who has this permission; v1 ships the annotation as the only halt-clearance mechanism (Phase 4 CLI wraps). |
| ProjectReconciler → init Job creation | Reconciler creates a Job that mounts the per-Project PVC. The init Job's SA is `tide-subagent` (zero K8s verbs — D-A4). |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-10-01 | Tampering | Operator setting bypass-budget to circumvent cap | accept | The annotation IS the documented operator override (D-D4); RBAC + K8s Events provide audit trail. |
| T-02-10-02 | Spoofing | Duplicate init Job created if reconcile races | mitigate | Deterministic name `tide-init-{project.UID}` + AlreadyExists handling. |
| T-02-10-03 | Denial of Service | PVC never binds → reconciler retries forever | accept | RequeueAfter 30s is unbounded; operator must fix the PVC. The reconciler doesn't crash. |
| T-02-10-04 | Information Disclosure | Init Job mounts the PVC → could read pre-existing files | mitigate | Pre-existing files are workspace state for THIS Project (no cross-Project access; PVC is per-Project per D-G1). Plan 04's subagent SA (tide-subagent) has zero K8s verbs so it can't enumerate other PVCs. |
| T-02-10-05 | Repudiation | Budget exceedance not logged | mitigate | K8s Event emitted with Reason=AbsoluteCapReached (Plan 03 added the ReasonBypassApplied + AbsoluteCapReached constants; reconciler records the Event). |
