<objective>
Wire B5 (tide-push clone provisions run branch + worktree) and B6 (controller passes --run-branch to clone job; PlanReconciler triggers TRUE per-wave integration before wave k+1 dispatch). Also adds the --integrate-task-branches pre-step to runPush for the final plan-boundary integration (D-04). These changes close SC-1, SC-3, SC-4, and SC-5.

Purpose: Without this plan, the run branch ref exists (B1 done) but is never called from production code; the run worktree is never provisioned so push mode fails; and most critically, wave k+1 executor worktrees fork from the run-branch tip at CLONE time — unless wave k-1 branches are integrated into the run branch BEFORE wave k+1 dispatches, dependents never see their dependencies' authored code (SC-3 violation, D-02 violation). Critically, "before dispatch" means the integration Job must COMPLETE (Status.Succeeded > 0), not merely be submitted — a running Job has not yet integrated anything into the run branch. A permanently-failed Job (BackoffLimit exhausted) must mark the Plan Failed immediately; continuing to requeue on Status.Succeeded==0 without checking Status.Failed creates a silent 5s livelock and violates the wave-boundary failure contract (dependents never dispatch; failure surfaces, not swallows).

Output: api/v1alpha1/plan_types.go (IntegratedThroughWave field), api/v1alpha1/shared_types.go (ReasonWaveIntegrationFailed constant), cmd/tide-push/main.go (runClone + runPush updated), internal/controller/push_helpers.go + push_helpers_test.go (CloneOptions.RunBranch + PushOptions.IntegrateTaskBranches), internal/controller/plan_controller.go (per-wave integration trigger + completion gate + failed-Job terminal path), internal/controller/boundary_push.go (integrateBranches threading), internal/controller/project_controller.go (CloneOptions.RunBranch wired).

  func (r *PlanReconciler) patchPlanFailed(ctx context.Context, plan *tideprojectv1alpha1.Plan, reason, message string) (ctrl.Result, error) {
      patch := client.MergeFrom(plan.DeepCopy())
      plan.Status.Phase = "Failed"
      meta.SetStatusCondition(&plan.Status.Conditions, metav1.Condition{
          Type:               tideprojectv1alpha1.ConditionFailed,
          Status:             metav1.ConditionTrue,
          Reason:             reason,
          Message:            message,
          LastTransitionTime: metav1.Now(),
      })
      if err := r.Status().Patch(ctx, plan, patch); err != nil {
          return ctrl.Result{}, err
      }
      return ctrl.Result{}, nil  // no requeue — terminal
  }
