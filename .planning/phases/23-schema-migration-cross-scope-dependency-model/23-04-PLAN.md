<objective>
Complete the v1alpha2 migration started by plans 23-01/02/03. Those plans made v1alpha2 the served+storage version and UNSERVED v1alpha1, but only migrated the webhooks + Wave controller. Every other consumer (the Project/Milestone/Phase/Plan/Task reconcilers, gates, dispatch, budget, reporter, CLI, dashboard, and ~80 test files) still imports `github.com/jsquirrelz/tide/api/v1alpha1`. As a result the reconcilers `For()`/`Owns()` v1alpha1 GVKs the API server no longer serves (the operator cannot function), and `go vet ./...` fails because `test/integration/envtest/suite_test.go` references the v1alpha1 webhook setup funcs that 23-02 moved to the v1alpha2 package.

This plan migrates ALL consumers to v1alpha2 so the operator runs on the served version, satisfying SCHEMA-03 ("no silent corruption / operator runs on the served version").

Purpose: un-break the operator at runtime. v1alpha1 GVKs are unserved; watching them is fatal. SCHEMA-03 demands the operator runs on the version it serves.

Output: every consumer compiles + vets on v1alpha2; no reconciler watches a v1alpha1 GVK; the envtest suite registers v1alpha2; the three semantic deltas (Wave ProjectRef, test SchemaRevision, webhook helper relocation) are resolved.
</objective>
