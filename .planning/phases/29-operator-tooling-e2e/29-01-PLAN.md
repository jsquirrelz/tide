<objective>
Build the shared `pkg/bundle/` package: the bundle on-disk format (types, tgz read/write,
sha256, childCount-stamp) and the offline dry-run validator. This is the pure, cluster-free
foundation both CLI verbs (export in 29-02, import/dry-run in 29-03) build on.

Purpose: A standalone Go package that `cmd/tide/` can import without dragging in
`internal/controller/` (which pulls full controller-runtime + envtest). It re-declares the
seed wire schema as a superset (adding `SHA256`), owns the zip-slip-safe tgz codec, and
houses the dry-run validation that reuses `pkg/dispatch` + `pkg/dag` client-side.

Output: `pkg/bundle/{seed,bundle,dryrun}.go` + their `_test.go` siblings, fully unit-tested
offline. No cluster, no pods, no new go.mod entries.
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| bundle file → CLI process | An attacker-influenceable .tgz is read/extracted by the operator's CLI; entry names + envelope bytes are untrusted input |
| seed manifest → dry-run graph | DependsOn refs in the bundle drive ComputeWaves; a crafted graph could attempt cycle/DoS |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-29-01-01 | Tampering | tgz extraction (bundle.go) | mitigate | Reject entries with `../` or absolute paths; filepath.Clean + dest-dir prefix check before any os.Create — task 2 zip-slip tests prove no write on either vector (severity: high) |
| T-29-01-02 | Tampering | out.json integrity (dryrun.go) | mitigate | sha256 compare against BundleEntry.SHA256; mismatch → fail-closed "re-plan", never adopt (D-04) (severity: medium) |
| T-29-01-03 | Tampering | seed DependsOn graph (dryrun.go) | mitigate | dag.ComputeWaves cycle detection; cycle → whole-import would-fail with InvolvedNodes, never partial adopt (D-09) (severity: medium) |
| T-29-01-04 | DoS | ComputeWaves on large graph | accept | Kahn is O(V+E); salvage is ~60 nodes; no untrusted unbounded fan-out (severity: low) |
| T-29-01-05 | Tampering | added `sha256` field corrupting seed | accept | Field is additive; ImportController json.Unmarshal drops unknowns (verified import_controller.go:311); no schema break (severity: low) |
