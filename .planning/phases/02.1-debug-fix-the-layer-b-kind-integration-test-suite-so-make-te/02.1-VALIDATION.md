## Sampling Rate

- **After every task commit:** Run `go build ./...` + relevant package tests (`go test ./internal/...` or `go test ./cmd/...` scoped to changed files)
- **After every plan wave:** Run `make test-int-layer-a` (fast envtest tier — catches regressions) — note: this Makefile target may be named `test-int-fast` instead; consult Makefile lines 126-128
- **Before `/gsd-verify-work`:** Full `make test-int` must be green (Layer A + all 7 Layer B specs) AND each run ≤ 300s
- **Max feedback latency:** 30s for unit + envtest; 5min for full kind tier
