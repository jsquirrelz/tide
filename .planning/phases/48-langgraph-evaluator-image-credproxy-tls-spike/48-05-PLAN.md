Verdict-classification detail for the script: `ssl.SSLCertVerificationError` / `httpx.ConnectError` at invoke = TLS FAIL (the thing being measured); an upstream HTTP-level auth error = TLS PASS with key problem (the TLS handshake necessarily succeeded to receive an HTTP response); a successful 1-token completion = full PASS.
