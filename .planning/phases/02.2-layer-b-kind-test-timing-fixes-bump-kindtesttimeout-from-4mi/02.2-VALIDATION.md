## Sampling Rate

- **After every task commit:** Run `go build ./test/integration/kind/... && go vet ./test/integration/kind/...` — compilation must not break.
- **After every plan wave:** Run `make test-int` from a clean kind state — full Layer A + Layer B must exit 0.
- **Before `/gsd-verify-work`:** Both `make test-int` (clean) AND `KEEP_KIND_CLUSTER=true make test-int` (rerun) must exit 0 with 7/7 Layer B specs PASS. Both runs ≤ 300s on the `go test` step (Makefile inner `timeout 300s` does not fire).
- **Max feedback latency:** ~300s (single `go test` wall-time budget; cold-cache prep is separate and unbounded).
