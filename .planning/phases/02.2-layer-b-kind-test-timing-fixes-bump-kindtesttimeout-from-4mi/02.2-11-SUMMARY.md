With cascade-8 closed (and now cascade-9A also closed — the prior failure mode at the Pod activeDeadline layer is gone), Pods finally get to the scheduling step in test namespaces; the PVC namespace gap blocks them there.

**Cascade-10 fix-shape proposal for Plan 02.2-12:** Option A — add `tide-projects` PersistentVolumeClaim declarations (RWO, matching cascade-1's chart contract) to each test fixture YAML in the same namespace as the test's Tasks. Smallest surface; preserves Option-B test-only contract; mirrors cascade-7's fixture-fix pattern from Plan 02.2-09.
