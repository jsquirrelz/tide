## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Backend JSON → React render | User-supplied CRD names render as React text nodes (escape-by-default). Zero dangerouslySetInnerHTML in any component file (T-04-D1 mitigation). |
| Clipboard write → operator terminal | CopyCommandButton writes UI-supplied commands via navigator.clipboard. The clipboard write itself is intentional — the user's terminal is the confirmation. No XSS surface here (we're writing OUT not IN). |

| T-04-D1-clipboard | Spoofing (clipboard hijack) | dashboard/web/src/components/CopyCommandButton.tsx | accept | navigator.clipboard requires user-initiated event (click); browser-platform guarantees this. Toast feedback gives user a chance to verify the copied text before pasting. |
