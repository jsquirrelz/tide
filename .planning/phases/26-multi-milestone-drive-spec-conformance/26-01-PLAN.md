This plan fixes all three so a Project drives multiple Milestones over ONE global Execution DAG.
Output: Multi-milestone template (+ golden/ratchet), §6d-free depgraph, N-safe idempotency guard, README planning-DAG-edge note, DEPS-02 reinterpretation record.
