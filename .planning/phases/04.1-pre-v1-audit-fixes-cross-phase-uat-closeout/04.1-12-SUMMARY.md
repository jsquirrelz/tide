## Root Cause of 11 Skips: kindTestTimeout Exhaustion

The `kindTestTimeout = 7 * time.Minute = 420s` context was exhausted by:
- BeforeSuite: 311.713s (helm upgrade --install with --wait --timeout 5m failed at 5m03s but the suite continued)
- chaos_resume spec: 16.356s
- output_test spec: 141.720s
- **Total: 469.789s > 420s**

After 420s, `context.WithTimeout` cancelled the context. Subsequent `BeforeEach` calls to `skipIfCRDsOnlyMode()` call `k8sClient.List(ctx, pl)` — with the cancelled context, this returns a context deadline exceeded error, triggering `Skip("CRDs not installed or controller not ready; skipping kind test")`.

The helm upgrade failed with `Error: UPGRADE FAILED: context deadline exceeded` because the suite context (420s total) had less than 300s remaining when helm started, but helm's `--wait --timeout 5m` tried to use 300s. This is a tight timing constraint that was previously acceptable but failed on this run.
